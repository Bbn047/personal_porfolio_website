# personal_porfolio_website

A responsive and interactive personal portfolio website built to showcase my profile, skills, education, experience, projects, certifications, and contact information.
The site is designed with a modern UI, smooth animations, and clean navigation to provide a professional online presence.


# Features

*Responsive Design

Fully responsive layout for desktop, tablet, and mobile devices

Mobile-friendly hamburger navigation menu

*Dynamic Navigation

Active section highlighting on scroll

Smooth scrolling between sections

*Home Section

Profile introduction with animated typing effect

Social media links (LinkedIn, Instagram, GitHub)

Resume download button

*About Section

Professional summary

Education and experience details with styled cards

*Certifications Section

Certification cards with ratings and hover effects

*Projects Section

Clickable project cards

Modal popup displaying detailed project information

*Contact Section

Contact form with client-side validation

Direct email and phone links

*UI Enhancements

Animated profile image

Hover effects and transitions

Modal animations for project details


# Technologies Used

HTML5 – Semantic page structure

CSS3 – Custom styling, variables, animations, and responsive layouts

JavaScript (ES6) – Interactivity and DOM manipulation

# How to Run Locally

Clone the repository:

git clone https://github.com/your-username/portfolio-website.git


Navigate to the project folder:

cd portfolio-website


Open index.html in your browser:

Double-click the file
OR

Use a live server extension (recommended)

# Customization

Update personal details in index.html

Replace images inside the images/ folder

Modify colors and layout using CSS variables in styles.css

Add or update projects in the Projects section
